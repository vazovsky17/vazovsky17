## Что я умею
- Проектирую архитектуру мобильных приложений (MVVM/MVI + Clean Architecture) → код проще поддерживать и масштабировать.
- Собираю UI на Jetpack Compose → быстрее вывожу фичи в прод без потери качества интерфейса.
- Внедряю офлайн/локальные слои данных (Room, кеширование) → приложение стабильно работает при плохой сети.
- Настраиваю сетевой слой и асинхронность (Retrofit/Ktor, Coroutines/Flow) → предсказуемое поведение и меньше edge-case багов.
- Довожу фичи до релиза: от идеи и UI до интеграций и поддержки → команда получает результат, а не «почти готово».
